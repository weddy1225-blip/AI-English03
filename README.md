# 🕵️ 校園偵探 (AI Campus Detective)

這是一個專為 **國小四年級學生** 設計的互動式英文學習遊戲。透過扮演偵探，學生需要在 AI 提供的校園情境線索中，運用聽力、閱讀與文法邏輯來破案。

## 🌟 遊戲特色

* **🎙️ TTS 即時語音：** 整合瀏覽器原生地美式英語發音，強化聽力連結。
* **🧩 智慧解謎邏輯：** 學生需根據 AI 的環境描述（例如：發現披薩盒）推理出正確的英文動作或喜好。
* **💡 輔助學習機制：** 提供「中英翻譯」與「隱藏式提示」，引導學生自主思考。
* **🏆 遊戲化計分系統：** 根據「破案時間」、「求助次數」與「答題正確率」自動結算積分與偵探等級（S/A/B）。
* **📱 友善介面：** 針對小學生優化的超大字體、可愛色調以及清楚的排版。

## 🎯 教學目標

1.  **單字辨識：** 區分拼寫相近的單字陷阱（例如：name/main, likes/lake）。
2.  **文法應用：** 練習第三人稱單數動詞變化（He likes...）與地點介系詞。
3.  **語感培養：** 透過反覆聽取美式口音，建立正確的語調印象。

## 🚀 如何開始遊戲

此專案為單一 HTML 檔案架構，無需安裝任何伺服器環境：

1.  下載本專案的 `index.html` 檔案。
2.  直接使用瀏覽器（推薦 Chrome 或 Edge）開啟檔案。
3.  點擊 **「聽聲音 (Listen)」** 開始調查案件！

## 📊 計分標準

遊戲採用 **「邏輯加減分」** 引擎：
* **基礎分：** 1000 分
* **答錯扣分：** 每錯一次扣 30 分。
* **求助扣分：** 查看提示扣 50 分，查看翻譯扣 20 分。
* **時間獎勵：** 破案速度愈快，額外加分愈多。

## 🛠️ 技術架構

* **Frontend:** HTML5, CSS3 (Flexbox 佈局)
* **Logic:** Vanilla JavaScript
* **Voice:** Web Speech API (SpeechSynthesis)
* **Deployment:** 適合部署於 GitHub Pages
