# 🕵️ 校園偵探：聽力推論任務 (AI Campus Detective)

這是一個專為 **國小三、四年級學生** 設計的互動式英文聽力學習遊戲。學生扮演偵探，在自動播放的證詞線索中，運用聽力與單字辨識能力來還原案件真相。

## 🌟 遊戲特色

*   **🎙️ 自動語音引導：** 進入題目時自動啟動美式英語發音，營造即時偵查的臨場感。
*   **📚 龐大案卷庫：** 內建 **50 題** 涵蓋生活單字、動物、天氣與日常動作的聽力測驗。
*   **🔄 智慧防重複機制：** 紀錄已完成的題目，確保學生在多輪遊玩中能完整練習 50 個不同的語言情境。
*   **💡 鷹架學習輔助：** 提供「顯示翻譯線索」功能，在學生遇到瓶頸時給予支持（會扣除少量積分）。
*   **📝 任務卷宗回顧：** 遊戲結束後顯示「總用時」並列出該場次的「答題紀錄與翻譯」，方便學生檢視錯誤。
*   **🏆 階級評等系統：** 根據最終積分自動評定為「首席神探 (S)」、「專業偵探 (A)」或「實習偵探 (B)」。

## 🎯 教學目標

1.  **聽力辨識：** 練習從完整的句子中聽取關鍵單字（Keyword Extraction）。
2.  **誘答排除：** 區分聽覺或拼寫相近的干擾選項（如：seven/six, Monday/Monkey）。
3.  **後設認知：** 透過結算頁面的題目回顧，強化單字記憶與自我校正能力。

## 🛠️ 如何啟動專案

此專案為單一 HTML 檔案架構，適合各種教學環境：

1.  下載本專案的 `index.html` 檔案。
2.  直接使用瀏覽器（推薦 Chrome、Edge 或行動裝置瀏覽器）開啟。
3.  進入題目後系統將**自動播放**錄音，若需重複聽取可點擊「播放證詞錄音」。

## 📊 計分標準

遊戲採用 **「動態積分」** 引擎：
*   **初始積分：** 1000 分。
*   **答錯懲罰：** 關鍵字選錯一次扣 **50 分**。
*   **查看翻譯：** 使用求助線索扣 **20 分**。
*   **用時顯示：** 於結算頁面顯示完成任務的總秒數，作為競賽參考。

## ⚙️ 技術架構

*   **Frontend:** HTML5, CSS3 (具備 RWD 自適應排版)
*   **Logic:** Vanilla JavaScript (ES6+)
*   **Voice:** Web Speech API (SpeechSynthesis)
*   **Storage:** LocalStorage (用於實現 50 題不重複機制)
*   **Deployment:** GitHub Pages
